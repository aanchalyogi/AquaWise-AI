# AquaWise AI

## Smart Water Conservation & Sustainability Advisor

AquaWise AI is a web-based sustainability project developed using Python, Flask, HTML, CSS, and Matplotlib.

### Features
- Water usage estimation
- Sustainability score
- Water efficiency grade
- Monthly water savings prediction
- AI-based recommendations
- Pie chart visualization
- SDG 6 aligned solution

### Technologies Used
- Python
- Flask
- HTML
- CSS
- Matplotlib

### SDGs Supported
- SDG 6: Clean Water and Sanitation
- SDG 11: Sustainable Cities and Communities
- SDG 12: Responsible Consumption and Production
- SDG 13: Climate Action
